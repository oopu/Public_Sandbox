Malware Events

Set up notifications to alert designated individuals when a malware event occurs. One or more of these protocols can be used for notifications: email (using SMTP), http/web servers, ryslog (remote syslog server), or SNMP servers.
  1. Configure operational mode settings.
  2. Create a comfort page.
  3. Run a deployment check.
  4. Set up notifications.
